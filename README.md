## ✨ Funcionalidades

- 🎨 **Design Premium** - Interface moderna, elegante e profissional
- 📱 **Totalmente Responsivo** - Perfeito em desktop, tablet e celular
- 🌙 **Dark Mode** - Tema escuro com toggle elegante
- ⚡ **Animações Suaves** - Framer Motion para transições fluidas
- 💬 **WhatsApp Flutuante** - Botão de contato rápido
- 🔍 **SEO Otimizado** - Meta tags, Open Graph e semântica HTML5
- 🖼️ **Galeria com Lightbox** - Visualização de imagens em tela cheia
- 📝 **Formulário Funcional** - Envio direto via WhatsApp
- 🗺️ **Google Maps** - Mapa integrado com localização
- 🚀 **Loading Elegante** - Tela de carregamento animada
- 🔄 **Scroll Suave** - Navegação entre seções

## 🚀 Como Executar

### Pré-requisitos

- **Node.js** 18+ instalado ([download](https://nodejs.org/))
- **npm** ou **yarn**

### Instalação

```bash
# 1. Acesse a pasta do projeto
cd giacomet-informatica

# 2. Instale as dependências
npm install

# 3. Execute em modo desenvolvimento
npm run dev

# 4. Acesse no navegador
# http://localhost:3000
```

### Build para Produção

```bash
npm run build
npm start
```
